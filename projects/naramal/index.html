플랫폼 Windows
장르 물리 기반 걷기 아케이드
개발 플랫폼 Unity
진행 기간 2020년 9월 ~ 2021년 10월
관련 링크 공개 소스

나라말은 .Net에서 유연한 한글 처리를 위해 만들어진 라이브러리입니다. 현재는 특별한 기능 추가나 유지보수는 계획되어 있지 않으나 추가적인 기능이 필요하다고 판단될 경우 업데이트 예정입니다.

1. 기능

1.1. 한글 자소 분리 및 조합

한글의 다양한 처리를 할 수 있도록 한글의 자소 분리 및 조합 기능을 구현하였습니다.

1.2. 조사 처리

단어에 따라 달라지는 한국어의 조사를 처리할 수 있도록 조사 처리 기능을 구현하였습니다. 문자열 포매팅 기능을 확장하여 구현하였으며 기존 문자열 포매팅 시스템을 확장하여 구성하였기 때문에 기존 포매팅 문자열 기반 위에서 동작할 수 있습니다.
